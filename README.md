# 🚀 Software Quality Assurance Project

Projeto de implantação de processo estruturado de testes de software.

Estruturar um processo completo de QA com foco em:

- Qualidade
- Rastreabilidade
- Gestão de Defeitos
- Cobertura de Testes
- Redução de Falhas
- Melhoria Contínua
